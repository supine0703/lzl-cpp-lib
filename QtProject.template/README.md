
# Qt Project Template

## 中文

这是我自用的基于 `CMake` 的 `Qt` 项目模板，包含常用的配置和封装，主要是为了方便快速创建 `Qt` 项目。
同时有一些第三方库的引入方法，比如 `spdlog`, `hidapi` 等。但没有包含源码，需要自行下载。
并且这些测试和使用主要在 `Windows` 平台，其他平台未进行测试。
除此之外，主项目常用在 `Windows` 和 `Linux` 平台，常用 `QtCreator` 和 `VSCode` 进行联合开发。

## English

This is my personal CMake-based Qt project template that includes common configurations and encapsulations, mainly to facilitate the rapid creation of Qt projects.
There are also methods to introduce some third-party libraries, such as `spdlog`, `hidapi`, etc. 
However, the source code is not included and needs to be downloaded separately.
These tests and uses are mainly on the `Windows` platform, and other platforms have not been tested.
In addition, the main project is commonly used on `Windows` and `Linux` platforms, often using `QtCreator` and `VSCode` for joint development.
